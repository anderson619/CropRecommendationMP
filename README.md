# 🌾 Crop Recommendation System (AI)

## 📖 Descripción del Proyecto
Este sistema es una solución de **Agricultura de Precisión** que utiliza modelos de Machine Learning para recomendar el cultivo ideal según las condiciones del entorno.

## 🤖 Modelos de Machine Learning
El sistema permite comparar y utilizar dos aproximaciones algorítmicas:
1. **Random Forest (RF)** 
2. **Support Vector Machine (SVM)**
